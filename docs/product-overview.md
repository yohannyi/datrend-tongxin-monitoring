# 产品与解决方案总览

## 公司简介
成都市大趋势同新材料有限公司是一家**高新技术企业**，专注于岩土工程与结构安全智能监测领域。

我们提供**设备 + 平台 + 售后**一体化解决方案，已服务西南地区水库、尾矿库、矿山等100多家单位。

## 核心监测设备

| 设备名称       | 主要功能                     | 适用场景                     |
|----------------|------------------------------|------------------------------|
| 渗压计        | 监测渗流压力                 | 水库大坝、尾矿库             |
| 测斜仪        | 监测深部水平位移             | 边坡、坝体                   |
| GNSS位移监测  | 高精度表面位移监测           | 滑坡、边坡、隧道             |
| 应力计        | 监测土体/结构应力变化        | 隧道、挡墙、大坝             |
| 测风仪        | 监测风速风向                 | 高边坡、露天矿山             |
| 多通道数采仪  | 多传感器数据采集与传输       | 各类安全监测系统             |
