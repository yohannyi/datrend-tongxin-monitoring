# 成功案例

## 案例1：四川省凉山州美姑县子威村中型水库大坝监测项目
- **项目类型**：水库大坝渗压、位移、应力综合监测
- **设备配置**：渗压计 + 测斜仪 + GNSS + 多通道数采仪
- **成果**：实现大坝全生命周期安全在线监测，获得业主高度评价。

## 案例2：深圳市球水山公园边坡滑坡监测
- **项目类型**：高陡边坡实时位移监测
- **设备配置**：GNSS位移 + 测斜仪
- **成果**：提前预警潜在风险，确保施工安全。

- **设备配置**：渗压计 + 应力计 + 多通道采集系统
